# BIBLIA.md - SARA

## Objetivo funcional
Convertir direccion en ejecucion sostenida mediante un sistema que registre, mida, analice y ajuste las distintas areas de vida para alcanzar objetivos reales de forma consistente.

## Naturaleza del producto
SARA es una app de uso personal. No tiene presion comercial ni usuarios externos que obliguen a conservar decisiones tempranas por compatibilidad.

Esto habilita iteracion rapida, refactors y cambios profundos cuando mejoren el sistema. La velocidad no habilita desorden: se mantienen arquitectura modular, tests, migraciones versionadas y trazabilidad completa.

## Regla madre
LAS DECISIONES SE TOMAN CON DATOS Y PROTOCOLOS, NO CON EL ESTADO EMOCIONAL DEL MOMENTO.

## Ley primera de construccion
NUNCA MEZCLAR RESPONSABILIDADES DE CAPAS.

Cada funcionalidad debe vivir en su modulo propietario, exponer contratos claros y tener pruebas propias. Agregar o modificar un modulo no debe cambiar el comportamiento de los modulos existentes salvo decision explicita, documentada y validada con tests.

## Que problema resuelve
- Evita que el conocimiento, las decisiones y los aprendizajes se pierdan en el tiempo.
- Convierte experiencia en progreso acumulativo.
- Reduce la distancia entre saber que hacer y hacerlo.

## Principios del sistema
- La acumulacion vale mas que la intensidad momentanea.
- Las tareas existen para avanzar objetivos.
- Los objetivos existen para avanzar planes.
- Los planes existen para avanzar una direccion.
- Una mala decision no invalida el plan.
- Las emociones se registran, pero no gobiernan.
- Todo lo importante debe quedar registrado.
- El sistema debe adaptarse a la realidad observada.
- La trazabilidad debe ser completa desde el dato de origen hasta la decision, ejecucion y cambio de estado.
- Cada modulo debe poder evolucionar sin acoplarse a internals de otros modulos.

## Areas iniciales
- Finanzas
- Negocios
- Salud
- Fisico
- Vida Futura

## Objetivo final
Construir un sistema operativo personal que ayude a ejecutar de forma consistente durante anos, reduciendo improvisacion, aumentando conciencia operativa y acelerando la consecucion de objetivos importantes.
