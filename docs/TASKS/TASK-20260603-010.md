## Intencion funcional

SARA debe poder recibir mensajes como:

```text
recordame en 5 minutos llamar al contador
```

Debe crear un recordatorio persistente y responder solo si la creacion fue real:

```text
Recordatorio creado para <fecha/hora>: llamar al contador
```

Luego, cuando llegue `due_at`, el dispatcher debe enviar por Chatwoot:

```text
Recordatorio: llamar al contador
```

Tambien debe soportar:

```text
que recordatorios tengo
cancelar recordatorio 1
```

### 7. Dispatcher de vencidos

Agregar dispatcher interno para recordatorios vencidos.

Responsabilidad:

- reclamar recordatorios `pending` con `due_at <= now()` solo para account `7`, inbox `45`, conversation `85`;
- pasarlos atomicamente a `processing`;
- enviar mensaje por Chatwoot a la conversacion del recordatorio;
- marcar `sent` si Chatwoot envio correctamente;
- marcar `failed` si Chatwoot falla;
- emitir eventos via RPCs.

Mensaje de disparo:

```text
Recordatorio: <title>
```

Guardas:

- No enviar si `status != processing`.
- No enviar si account/inbox/conversation no es `7/45/85` en este MVP.
- No enviar recordatorios cancelados.
- No duplicar envios.
- Si falla Chatwoot, registrar fallo; no marcar `sent`.

Frecuencia sugerida:

- intervalo configurable por env var o default seguro de 30 segundos.

No bloquear el worker post-buffer si falla dispatcher.

### 8. Integracion con session-context

Despues de acciones ejecutadas:

`reminders.create`:

- foco: `focusedEntityType = "reminder"`
- `focusedEntityId = reminderId`
- `activeModule = "reminders"`
- `activeFlow = "reminder_created"`
- `context.lastReminderTitle = title`
- `context.lastReminderDueAt = dueAt`

`reminders.list`:

- si hay un unico recordatorio pendiente, foco en ese recordatorio.
- si hay multiples, guardar `context.lastReminderList` con posiciones e ids.

`reminders.cancel`:

- limpiar foco si el recordatorio enfocado fue cancelado.
- `activeFlow = "reminder_canceled"`.

Resolver referencias simples solo si no hay ambiguedad:

- `cancelar ese`
- `cancelar el ultimo recordatorio`

No usar LLM para resolver referencias.
