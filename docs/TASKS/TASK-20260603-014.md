## Parser MVP

Debe soportar:

- `crear rutina manana normal: 7:00 despertar; 7:15 desayuno`
- `crear rutina manana normal`
- `listar rutinas`
- `activar rutina manana normal`
- `pausar rutina manana normal`
- `archivar rutina manana normal`

Reglas:

- Slug deterministico ASCII-safe.
- Si create no trae nombre, `missingData = ["routineName"]`.
- Si step trae hora invalida, `missingData = ["routineStepTime"]`.
- No crear recordatorios ni tareas automaticas.
