## Parser MVP

Debe soportar:

- `arrancar gym`
- `arrancar gym piernas`
- `sentadilla serie 1 8 reps 60kg`
- `listo sentadilla 8 reps 60kg`
- `terminar gym`
- `cancelar gym`
- `que entrenamientos tengo`

Reglas:

- Si log-set no trae ejercicio: `missingData = ["exerciseName"]`.
- Si log-set no trae reps ni duracion: `missingData = ["workoutEffort"]`.
- Si no hay sesion activa resoluble, no ejecutar log-set.
- No inventar peso, reps ni duracion.
