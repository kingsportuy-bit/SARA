
## Parser MVP

Debe soportar:

- `crear protocolo energia baja: si dormi menos de 6 horas, rutina liviana`
- `que protocolos tengo`
- `activar protocolo energia baja`
- `archivar protocolo energia baja`
- `evaluar protocolo energia baja`

Reglas:

- Las reglas se guardan como texto estructurado simple en JSON.
- Evaluar no ejecuta acciones.
- Evaluar puede devolver sugerencias textuales con evidencia, pero no mutar otros modulos.
- No usar LLM para decidir.
