## Parser MVP

Debe soportar:

- `crear plan mejorar energia: caminar 20 minutos; dormir antes de 23`
- `crear plan para objetivo mejorar energia: caminar; dormir`
- `que planes tengo`
- `archivar plan mejorar energia`
- `completar paso 1 del plan mejorar energia`

Reglas:

- No generar pasos si el usuario no los dio.
- No crear tareas automaticamente.
- Si falta titulo de plan, `missingData = ["planTitle"]`.
- Si falta step en create, `missingData = ["planSteps"]`.
