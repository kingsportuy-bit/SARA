# COMMUNICATION_PROTOCOL.md - SARA

## Objetivo
Estandarizar handoff, bloqueos y cierres entre agentes.

## Formato de entrega
- Fecha (UTC)
- Task ID
- Estado
- Cambios
- Evidencia
- Riesgos
- Siguiente paso

## Formato de bloqueo
- Fecha (UTC)
- Task ID
- Bloqueo
- Impacto
- Accion requerida del operador/orquestador

## Regla
No hay cambio valido sin task y evidencia trazable.
